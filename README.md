 Fibonacci Generator

This project is a Python-based Fibonacci Series Generator developed as part of my internship at HexSoftwares.

The program generates the Fibonacci sequence up to the number specified by the user.

---

 Features

✔ Takes user input  
✔ Generates Fibonacci sequence  
✔ Input validation  
✔ Clean and simple terminal interface  

---

 Technologies Used

- Python 3.10
- Loops
- Conditional Statements
- Lists

 How It Works

The Fibonacci series is a sequence where:

Each number = Sum of previous two numbers

Example:

Input: 7

Output:
0, 1, 1, 2, 3, 5, 8
